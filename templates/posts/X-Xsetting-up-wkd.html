Setting up WKD

There are not a lot of guides and knowledge bases for WKD, so I decided to write what I learned while setting it up. I will be using m3r.one as an example domain, but you should use yours.

What is WKD?

To start from be beginning, WKD is a way for email clients or tools such as GPG to retrieve public keys with associated email addresses.

Setting up WKD

There are 2 methods with fetching/setting up from WKD: advanced and direct. When fetching, clients first must look at the advanced method URL's and then they should look at the direct method. However, sometimes clients may not adhere to the standards, so we'll go over setting up both of these methods. Both of these methods require an empty file called policy to reside in a directory.

Advanced Method

For the advanced method, there first has to be a subdomain called openpgpkey, so this method requires access to editing DNS records a domain.
